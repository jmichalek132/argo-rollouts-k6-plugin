## Tech Debt

### Phase 4: Release & Examples

1. **e2e workflow missing `kind` install** — `.github/workflows/e2e.yml` does not install the `kind` binary; ubuntu-latest does not include it. Fix: add `go install sigs.k8s.io/kind@latest` or use `helm/kind-action@v1`.

2. **e2e workflow missing `-timeout=15m`** — Go's default 10-minute test timeout will kill the e2e suite (~12-15 min). Makefile's `test-e2e` target has `-timeout=15m` but the CI workflow does not.

3. **Phase 4 human verification items** — 3 checks require live environment: e2e test execution against kind cluster, GitHub Release artifact verification, e2e GitHub Actions runner confirmation.

### Phase 3: Step Plugin

4. **STEP-03 JSON key naming** — `RpcStepResult.Status` JSON uses `"runId"` instead of `"testRunId"`. No functional impact (controller treats Status as opaque bytes), but misaligns with requirement wording. Consider renaming or updating requirement text.
